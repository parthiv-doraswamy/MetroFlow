# MetroFlow – Smart Route Planner

MetroFlow is a full-stack graph-based metro route planner.

It integrates:

- C++ graph algorithms
- Node.js backend
- Interactive frontend visualization

---

# Features

- Dijkstra Shortest Path
- BFS Traversal Visualization
- Responsive UI
- Graph Visualization
- Distance Calculation
- Estimated Travel Time
- REST API Integration

---

# Tech Stack

## Frontend

- HTML
- CSS
- JavaScript
- Vis.js

## Backend

- Node.js
- Express.js

## Algorithms

- C++
- STL
- Graph Algorithms
- Priority Queue
- Adjacency List

---

# Installation

## Install Node Modules

```bash
cd backend
npm install
```

---

# Compile C++

```bash
cd cpp
g++ metro.cpp -o metro.exe
```

---

# Start Backend

```bash
cd backend
npm start
```

---

# Run Frontend

Open:

```text
frontend/index.html
```

# Resume Description

MetroFlow – Smart Route Planner
C++, Node.js, JavaScript, Graph Algorithms, HTML, CSS

- Built a full-stack metro route planner integrating a Node.js backend with C++ graph algorithms.
- Implemented Dijkstra’s Algorithm and BFS in C++ using STL, priority queues, and adjacency lists.
- Developed REST APIs in Express.js to execute native C++ route computation dynamically.
- Designed an interactive frontend visualization system using JavaScript and Vis.js for route mapping and traversal animation.
- Optimized shortest-path computation using scalable graph architecture and efficient pathfinding logic.
